# 🏥 Healthcare Data Analysis Dashboard (Power BI Project)

## 🔍 Project Overview

This project presents an interactive Power BI dashboard built using healthcare data. It provides insights into patient records, hospital performance, and treatment trends to support data-driven decision-making in the healthcare domain.

## 🎯 Objectives

* Analyze patient data and hospital performance
* Track admission and discharge trends
* Identify high-demand treatments and departments
* Monitor overall healthcare efficiency

## 🛠️ Tools & Technologies Used

* Power BI
* Microsoft Excel (Dataset)
* Data Cleaning & Transformation
* DAX

## 📊 Dashboard Features

* Interactive filters and slicers
* Patient admission and discharge analysis
* Department-wise performance
* Treatment and diagnosis insights
* Time-based trend analysis

## 📈 Key Insights

* Identified peak admission periods
* Highlighted high-demand medical departments
* Observed patterns in patient treatment trends
* Improved visibility into hospital performance

## 📁 Files Included

* `Healthcare Power BI.pbix` (Dashboard)
* `Hospital Health Care Management Data set.xlsx` (Dataset)

## 🚀 How to Use

1. Download both files
2. Open `.pbix` file in Power BI Desktop
3. Reconnect dataset if needed
4. Explore dashboard using filters

## 💡 Conclusion

This project demonstrates the application of Power BI in the healthcare domain, transforming raw data into actionable insights for improving hospital operations and patient care.
